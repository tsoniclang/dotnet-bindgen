# dotnet-bindgen

Generate TypeScript declarations from .NET assemblies.

This is a convenience wrapper for [@tsonic/dotnet-bindgen](https://www.npmjs.com/package/@tsonic/dotnet-bindgen).

## Installation

```bash
npm install dotnet-bindgen
```

## Usage

```bash
npx dotnet-bindgen generate -a MyAssembly.dll -o out/
```

## Requirements

- .NET 10 runtime

## Documentation

See the full documentation at [@tsonic/dotnet-bindgen](https://github.com/tsoniclang/dotnet-bindgen).
